### Prerequisites
- node (https://nodejs.org/en/)
- git
- PostgresSQL
- MinIO (or any S3 service)


#### How to build 
```bash
$ npm run build
```

#### How to run in dev
```bash
$ npm run dev
```

#### How to run in production
```bash
$ npm run start
```
